## What to build

The Archivist prompt marks anonymous placeholders using a local `ANONYMOUS_MARKERS` array in `buildArchivistTurnMessage`. Apply-patch/reference-policy has its own anonymity detection path. These can drift, and the current marker list is genre-thin (`unknown`, `young man`, `girl`, `elder`, `aide`).

Move anonymous placeholder detection into one shared policy module with a base list plus genre-specific additions. Use it both when rendering the Archivist registry and when resolving NPC merges.
