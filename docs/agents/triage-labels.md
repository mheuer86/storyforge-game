# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the local markdown status strings used in this repo.

| Label in mattpocock/skills | Label in our tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an AFK agent |
| `ready-for-human` | `ready-for-human` | Requires human oversight or judgment |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a role, use the corresponding status string from this table.
