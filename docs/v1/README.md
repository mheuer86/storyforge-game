# V1 Documentation Snapshot

This folder preserves the pre-SF2-primary documentation set. Most files describe the V1 game that now lives at `/play/v1`:

- `app/api/game/route.ts`
- `components/game/*`
- `lib/system-prompt.ts`
- `lib/tools.ts`
- `lib/tool-processor.ts`

Use the root `docs/` files for the current `/play` SF2 game.
